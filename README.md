# SyntaxWear - E-commerce de Tênis e Sneakers

## Descrição

SyntaxWear é um site de e-commerce dedicado à venda de tênis e sneakers online. O site oferece uma experiência de compra intuitiva com seções para produtos masculinos, femininos e outlet, além de informações sobre lojas físicas e online. Desenvolvido com HTML e CSS puro, o site é responsivo e acessível.

## Funcionalidades

- **Página Inicial**: Seção hero com destaque para o modelo "Krypton One", botões de ação para ver modelos e comprar.
- **Categorias**: Navegação por categorias como Casual, Esporte, Moderno e Futurista.
- **Produtos**: Grid de produtos em destaque com títulos, subtítulos e links para gêneros.
- **Newsletter**: Formulário para inscrição em e-mails.
- **Redes Sociais**: Links para Instagram, WhatsApp, TikTok e Facebook.
- **Footer**: Links organizados por seções (Masculino, Feminino, Outlet, Lojas, Sobre).
- **Responsividade**: Design adaptável para dispositivos móveis e desktop.

## Tecnologias Utilizadas

- **HTML5**: Estrutura semântica da página.
- **CSS3**: Estilização com variáveis CSS para consistência, reset, base e estilos específicos.
- **Imagens e Ícones**: SVGs e JPGs para logos, ícones e produtos.

## Como Executar

1. Clone o repositório:
   ```
   git clone https://github.com/Guigagemes/ecommerce-syntaxwear.git
   ```

2. Navegue até o diretório do projeto:
   ```
   cd ecommerce-syntaxwear
   ```

3. Abra o arquivo `index.html` em um navegador web.

Como é um site estático, não há necessidade de servidor ou build. Basta abrir o arquivo HTML diretamente.
